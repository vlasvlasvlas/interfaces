# Interfaces

Galeria simple de interfaces visuales, sonoras e interactivas de `vlasvlasvlas`.

Los proyectos se editan en `data/interfaces.yaml`. La pagina carga ese YAML y renderiza tarjetas con descripcion, tags, link al repo, demo si existe e imagen si esta declarada.

## GitHub Pages

Cuando GitHub Pages este habilitado para la rama `main`, la galeria queda disponible en:

https://vlasvlasvlas.github.io/interfaces/

## Desarrollo local

```sh
python3 -m http.server 4173
```

Luego abrir `http://localhost:4173`.
